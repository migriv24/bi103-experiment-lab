# Strength Into Speed — Experimental Design Media

BI 103 Science Skills · Oregon State University

A designed (not conducted) experiment: **does adding two lower-body strength sessions per
week make my easy runs faster?** Presented as a blueprint-style lab protocol, grounded in
the finding that strength training improves running economy.

This ties together three other labs: the Skeletal & Muscular Lab (the muscles trained),
Soft Running / Body Data (the pace measured), and the Data Management Lab (how the data
would be collected).

## Status

Shell build. The illustrations are **placeholders**: each figure shows a dashed box with
the target filename. Drop the real hand-drawn art into `images/` using those filenames and
the placeholders are replaced automatically (the `<img>` tags swap in on load).

See `ASSETS.md` for the drawing checklist.
